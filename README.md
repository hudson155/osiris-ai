# Osiris AI

Osiris AI is a Kotlin OpenAI wrapper.

## Project information

### Major dependencies

- Gradle 8.10
- Kotlin 2.0
- Java 21
- Kairo 0.77

### Style guide

Please follow the [Kairo style guide](https://github.com/hudson155/kairo/blob/main/docs/style-guide.md).

## Getting started

**Run checks (tests and lint):**

```shell
./gradlew check
```

**Full build:**

```shell
./gradlew build
```

**Publish:**

```shell
./gradlew publish
```
